독립 계정이라면: 계정별 User OAuth(authorization_code) + Refresh Token DB 저장 + 1시간 갱신 로직 필수. 이 경우 Admin 화면 및 백엔드 복잡도 대폭 증가.
→ 이 전제를 먼저 확인하고 Admin 토큰 관리 화면 범위를 확정할 것.
